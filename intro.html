JSON, API and GET Requests

First, what is JSON?

JSON stands for Javascript Object Notation. It is a standard for retrieving key/value pairs.

For example, if we make a API GET Request for the first name and last name of, say everyone living in the same US Zip code, the result would be a data set that would look be in a format easily understood by JavaScript, as well as other languages and frameworks

All GET requests in this guide will be working with JSON requests.

BreweryDB, What does it do?

BreweryDB maintains an extensive database regarding beer and all things brewery.

This includes, but is not limited to:

  • Beers
  • Styles of Beer
  • Breweries
  • Categories of Beers
  • Ingredients
  • Locations where Beers are Served

What is mentioned above would be, said loosely, top level search fields. Within each of those categories listed above, one can further search for beers by other characteristics.

For example, if someone was looking for a specific beer that had a specific style ( IPA, Lager, Stout ) and ABV ( alcohol by volume ), one could construct an API GET request to do so

In the next section, we will explore this further by using a GET request with BreweryDB to list Beers meeting a specific set of parameters.
